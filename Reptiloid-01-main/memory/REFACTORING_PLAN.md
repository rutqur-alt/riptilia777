## Роли в системе:
1. **Трейдер** - продаёт USDT, управляет офферами и реквизитами
2. **Мерчант** - принимает платежи через API
3. **Админ (owner)** - полный доступ
4. **Модератор P2P (mod_p2p)** - решает споры
5. **Клиент мерчанта** - покупает через магазин (без регистрации)
